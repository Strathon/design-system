# Strathon Design System v1.0

Welcome to the **Strathon Design System** - the foundational layer for all Strathon AI Trust Layer interfaces. This design system provides enterprise-grade UI components, design tokens, and patterns that ensure consistency, accessibility, and performance across our platform.

## 🎯 **Purpose**

The Strathon Design System serves as our **Visual Constitution**, ensuring:

- **Consistency**: Unified experience across Pulse, Compass, Forge, and future interfaces
- **Accessibility**: WCAG 2.1 AA compliance built into every component
- **Performance**: Optimized components with <500KB bundle size
- **Developer Experience**: TypeScript-first with comprehensive documentation
- **Enterprise Quality**: Professional interfaces that build customer trust

## 🏗️ **Architecture**

Our design system follows a modular architecture:

```
@strathon/design-system/
├── @strathon/tokens     # Design tokens (colors, typography, spacing)
├── @strathon/ui         # React components
├── @strathon/icons      # SVG icon library
└── @strathon/utils      # Shared utilities
```

## 🎨 **Design Principles**

### 1. **Clarity Above All**
Information hierarchy is clear and scannable. Critical information is accessible within 3 seconds.

### 2. **Trust by Default**
Professional visual design and consistent interactions build confidence in our AI governance platform.

### 3. **Accessibility First**
Every component meets WCAG 2.1 AA standards with proper keyboard navigation and screen reader support.

### 4. **Performance Optimized**
Components are built for speed with tree shaking, lazy loading, and minimal runtime overhead.

### 5. **Context Engineering**
Components adapt to different contexts while maintaining consistency.

## 🚀 **Getting Started**

### Installation

```bash
npm install @strathon/ui @strathon/tokens
```

### Basic Usage

```tsx
import { Button, Card, Badge } from '@strathon/ui';

function App() {
  return (
    <Card>
      <h1>Welcome to Strathon</h1>
      <Badge variant="success">Active</Badge>
      <Button variant="default">Get Started</Button>
    </Card>
  );
}
```

### Theme Setup

```tsx
import '@strathon/tokens/dist/css/tokens.css';

function App() {
  return (
    <div data-theme="light"> {/* or "dark" */}
      {/* Your app content */}
    </div>
  );
}
```

## 🔧 **Development**

This design system is built with:

- **React 18+** with TypeScript
- **Tailwind CSS** for styling
- **Radix UI** for accessible primitives
- **Class Variance Authority** for component variants
- **Storybook** for documentation
- **Jest + Testing Library** for testing
